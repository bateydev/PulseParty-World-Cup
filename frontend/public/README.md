# PWA Assets

This directory contains the Progressive Web App assets for PulseParty Rooms.

## Icon Files

The following icon files are required for the PWA:

- `pwa-192x192.png` - 192x192 icon for Android devices
- `pwa-512x512.png` - 512x512 icon for Android devices and splash screens
- `apple-touch-icon.png` - 180x180 icon for iOS devices
- `favicon.ico` - Browser favicon

## Generating Icons

### Option 1: Using the SVG source

The `pwa-icon.svg` file contains the source icon design. You can convert it to PNG using:

**Using ImageMagick:**
```bash
# Install ImageMagick if not already installed
# Ubuntu/Debian: sudo apt-get install imagemagick
# macOS: brew install imagemagick

# Generate icons
convert -background none pwa-icon.svg -resize 192x192 pwa-192x192.png
convert -background none pwa-icon.svg -resize 512x512 pwa-512x512.png
convert -background none pwa-icon.svg -resize 180x180 apple-touch-icon.png
convert -background none pwa-icon.svg -resize 32x32 favicon.ico
```

**Using Inkscape:**
```bash
# Install Inkscape if not already installed
# Ubuntu/Debian: sudo apt-get install inkscape
# macOS: brew install inkscape

# Generate icons
inkscape pwa-icon.svg --export-filename=pwa-192x192.png --export-width=192 --export-height=192
inkscape pwa-icon.svg --export-filename=pwa-512x512.png --export-width=512 --export-height=512
inkscape pwa-icon.svg --export-filename=apple-touch-icon.png --export-width=180 --export-height=180
inkscape pwa-icon.svg --export-filename=favicon.ico --export-width=32 --export-height=32
```

**Using online tools:**
- Visit https://realfavicongenerator.net/
- Upload the `pwa-icon.svg` file
- Download the generated icons

### Option 2: Using the HTML generator

Open `icon-generator.html` in a browser, right-click on each canvas, and save the images with the appropriate filenames.

## Placeholder Icons

For development purposes, placeholder PNG files are included. Replace these with properly designed icons before production deployment.

## Manifest

The `manifest.webmanifest` file contains the PWA configuration. It's automatically generated by vite-plugin-pwa, but a static version is included here for reference.

## Theme Colors

- Primary theme color: `#1e40af` (blue-800)
- Background color: `#ffffff` (white)
- Accent color: `#fbbf24` (yellow-400)
